# 📱 KeenKeeper - Your Personal Friendship Tracker

A modern web application built with **Next.js** designed to help users maintain and nurture their relationships through consistent check-ins and interaction analytics.

---

## 🛠️ Technologies Used

- **Framework:** [Next.js 15+](https://nextjs.org/) (App Router, SSR)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) & [DaisyUI](https://daisyui.com/)
- **Charts:** [Recharts](https://recharts.org/) (for data visualization)
- **Icons:** [React Icons](https://react-icons.github.io/react-icons/)
- **State Management:** React Context API
- **Notifications:** React Toastify

---

## 🚀 Key Features

### 1. **Smart Relationship Management (SSR)**

The application uses Server-Side Rendering (SSR) to display friend profiles and details efficiently. It calculates the "Days Since Contact" and automatically tracks "Next Due" dates to ensure you never lose touch with your loved ones.

### 2. **Interactive Interaction Logs**

Users can record their interactions (Call, Text, or Video) with a single click. Every action is instantly updated in the history and triggers beautiful toast notifications, providing a seamless user experience.

### 3. **Dynamic Friendship Analytics**

Features a visually stunning Dashboard with **Donut Charts** (using Recharts). It provides a breakdown of interaction types, helping users visualize their communication habits and focus on areas that need more attention.
